Why?

One of my favorite photographers Darren Pearson (Dariustwin) specializes in light painting. The images shown are taken by him. I was inspired by his work and wanted to try it out myself. I had a lot of fun making this project and I hope you enjoy it!

What the Pros Use

Darren Pearson uses his own "Night-Writer Pro" to help him take these amazing photos. Because of the cost of one of these devices, I decided to make my own. I used a couple NeoPixels and a Seed Studio XIAO to make my own. I also used the same keyboard style switches to pay homage to the original device.

My "Light Painter" uses 3 3w RGBW LEDs instead of an array of single color LEDs. This allows for more color options and a wider range of brightness. The XIAO is a microcontroller that is capable of running the code needed to control the LEDs (on a side note, the XIAO is also capable of acting as a USB HID device, which allows it to act as a keyboard).

What Went Wrong?
v1↦v2

v1 of the light painter was a great success! Except that it didn't work. I had a lot of trouble getting the charlieplexing working for the indicator lights, this was the result of forgetting a wire on the schematic. This was fixed with an external wire taped onto the board.

Another issue with v1 was that I swapped pad 3 and 4 of the NeoPixels on my part footprint. This didn't have an easy fix like the previous problem, and required a new PCB to be manufactured.

The final issue with v1 was that the mouse bites to make the corner break off didn't work. This was fixed by using a dremel to cut the corner off.
